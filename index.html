Estilos de Lista: Tipo

Cambia el marcador de la lista (bullet point o número) con list-style-type.

ul {
  /* Círculos, cuadrados, discos... */
  list-style-type: square;
}

ol {
  /* Números, letras, romanos... */
  list-style-type: upper-roman; /* I, II, III */
  list-style-type: lower-alpha; /* a, b, c */
}
2 / 15

Posición del Marcador

list-style-position decide si la viñeta está dentro o fuera de la caja de contenido.

li {
  /* (Por defecto) La viñeta está fuera del flujo de texto */
  list-style-position: outside;

  /* La viñeta está dentro, el texto la envuelve */
  list-style-position: inside;
}
3 / 15

Imagen como Viñeta

Puedes usar una imagen personalizada en lugar de los puntos estándar.

ul {
  list-style-image: url('icono-check.svg');
}

/* Nota: Es difícil controlar el tamaño de esta imagen.
   A menudo es mejor usar ::before para iconos custom. */

Eliminar Estilos (Reset)

Esencial para crear barras de navegación y menús limpios.

nav ul {
  /* Quita los puntos/números */
  list-style: none;
  
  /* Quita el sangrado (indentación) por defecto */
  padding-left: 0;
  margin: 0;
}
5 / 15

Shorthand: List Style

Resume las tres propiedades en una sola línea.

ul {
  /* type | position | image */
  list-style: square inside url('img.png');
  
  /* Uso común para resetear */
  list-style: none;
}

Tablas en CSS

Las tablas HTML por defecto son feas y tienen doble borde. CSS es obligatorio para que se vean profesionales.

table {
  width: 100%;
  text-align: left;
}

th, td {
  padding: 10px;
  border: 1px solid #ddd;
}
7 / 15

Border Collapse

La propiedad más importante para tablas. Decide si las celdas comparten bordes o están separadas.

table {
  /* Las celdas tienen espacios entre ellas (Old school) */
  border-collapse: separate;

  /* Los bordes se fusionan en una sola línea (Moderno) */
  border-collapse: collapse;
}
8 / 15

Border Spacing

Si usas border-collapse: separate, puedes controlar el espacio entre celdas.

table {
  border-collapse: separate;
  
  /* Espacio Horizontal | Espacio Vertical */
  border-spacing: 10px 20px;
}
9 / 15

Table Layout

Controla cómo se calcula el ancho de las columnas. Vital para performance en tablas grandes.

table {
  /* El ancho depende del contenido (Lento, puede saltar) */
  table-layout: auto;

  /* El ancho es fijo/respetado. Más rápido y predecible. */
  table-layout: fixed;
  width: 100%;
}
10 / 15

Caption Side

Posiciona el título de la tabla (elemento <caption>).

caption {
  caption-side: bottom; /* Pone el título abajo */
  font-style: italic;
  padding: 10px;
}
11 / 15

Celdas Vacías (Empty Cells)

Controla si se muestran los bordes y fondo de celdas sin contenido.

table {
  border-collapse: separate;
  
  /* show (defecto) | hide */
  empty-cells: hide;
}
12 / 15

Diseño "Cebra" (Striped)

Mejora la legibilidad coloreando filas alternas usando pseudo-clases.

/* Aplica a las filas pares */
tr:nth-child(even) {
  background-color: #f2f2f2;
}

/* O usa (odd) para impares */
13 / 15

Efecto Hover en Filas

Ayuda al usuario a seguir la fila que está leyendo.

tr:hover {
  background-color: #ddd;
  cursor: pointer;
}
14 / 15

Resumen

Claves para listas y tablas modernas.

// Listas:
- Usa 'list-style: none' para menús.
- Controla la sangría con padding en el <ul>.

// Tablas:
- Siempre usa 'border-collapse: collapse'.
- Usa 'nth-child' para diseño de cebra.
- Envuelve la tabla en un div con 'overflow-x: auto'
  para que sea responsiva en móviles.
